Helm Upgrade:
- Modifies existing release
- Creates new revision

Helm History:
- Shows release revisions

Helm Rollback:
- Reverts to previous revision

Example:

Revision 1 -> Install
Revision 2 -> Upgrade
Revision 3 -> Upgrade
Revision 4 -> Rollback to Revision 2

Production Use:
- Recover from bad deployment
- Restore working configuration
- Quick rollback during incidents
